# キャラクターシステム機能仕様書

**最終更新**: 2026-01-20

---

## 概要

キャラクターのステータス・感情・成長を管理するデータクラス群。

---

## ファイル構成

| ファイル | 役割 |
|---------|------|
| `CharacterData.cs` | キャラクター基本データ（ScriptableObject） |
| `StatusParameter.cs` | ステータス計算・バフ管理 |
| `EmotionParameter.cs` | 感情状態の管理 |
| `EmotionEffect.cs` | 感情効果の倍率定義 |

---

## CharacterData.cs

### クラス概要

キャラクターの固定データを保持する ScriptableObject。

### フィールド

| フィールド | 型 | 説明 |
|-----------|-----|------|
| `characterId` | string | キャラクターID |
| `characterName` | string | 表示名 |
| `isBeast` | bool | 獣フラグ（恐怖時の効果が異なる） |
| `baseStatus` | StatusParameter | 基本ステータス |
| `growthRate` | StatusParameter | 成長率 |

---

## StatusParameter.cs

### クラス概要

HP, MP, 攻撃力などのステータス値とバフ状態を管理。

### フィールド

| フィールド | 型 | 説明 |
|-----------|-----|------|
| `maxHP`, `maxMP` | int | 最大HP/MP |
| `attack`, `magic` | int | 攻撃力/魔力 |
| `defense`, `magicDefense` | int | 防御力/抗魔力 |
| `speed` | int | 敏捷性 |

### バフ倍率

| フィールド | デフォルト | 説明 |
|-----------|-----------|------|
| `buffAttack` | 1.0f | 攻撃倍率 |
| `buffDefense` | 1.0f | 防御倍率 |
| `buffMagic` | 1.0f | 魔力倍率 |
| `buffMagicDefense` | 1.0f | 抗魔倍率 |
| `buffSpeed` | 1.0f | 敏捷倍率 |

### メソッド

| メソッド | 説明 |
|---------|------|
| `GetFinalAttack()` | バフ適用後の最終攻撃力 |
| `GetFinalDefense()` | バフ適用後の最終防御力 |
| `ResetBuffs()` | バフをリセット |
| `CalculateLevelStatus(level, growth)` | レベル成長計算 |

---

## EmotionParameter.cs

### クラス概要

現在の感情状態を管理。

### 感情タイプ

| 感情 | 効果 |
|------|------|
| `Joy`（喜び） | 全能力 +5% |
| `Anger`（怒り） | 攻撃 +15% |
| `Sadness`（哀しみ） | 魔力 +15% |
| `Fear`（恐怖） | 攻撃・魔力 -15%（獣は回避 1.5倍） |

### フィールド

| フィールド | 型 | 説明 |
|-----------|-----|------|
| `currentEmotion` | EmotionType | 現在の感情 |

### メソッド

| メソッド | 説明 |
|---------|------|
| `SetEmotion(type)` | 感情を設定 |
| `ClearEmotion()` | 感情をリセット |
| `GetEmotionEffect()` | 現在の感情効果を取得 |

---

## EmotionEffect.cs

### クラス概要

感情ごとの倍率を定義する構造体。

### フィールド

| フィールド | 型 | 説明 |
|-----------|-----|------|
| `attackMult` | float | 攻撃倍率 |
| `magicMult` | float | 魔力倍率 |
| `defenseMult` | float | 防御倍率 |
| `evasionMult` | float | 回避倍率 |
