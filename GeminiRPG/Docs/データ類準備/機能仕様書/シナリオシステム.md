# シナリオシステム機能仕様書

**最終更新**: 2026-01-20

---

## 概要

シナリオシステムは、CSVファイルからストーリーデータを読み込み、テキスト表示・選択肢・フラグ管理・画面演出などを順次実行するシステムです。

---

## ファイル構成

| ファイル | 役割 |
|---------|------|
| `ScenarioCommand.cs` | CSV1行分のデータを保持するクラス |
| `ScenarioLoader.cs` | CSVファイルをパースしてコマンドリストを生成 |
| `ScenarioExecutor.cs` | コマンドを順次実行するメインエンジン |

---

## ScenarioCommand.cs

### クラス概要

シナリオCSVの1行分のデータを保持するデータクラス。

### フィールド（5列方式）

| フィールド | 型 | 説明 |
|-----------|-----|------|
| `Label` | string | ジャンプ先ラベル（例: "Ev_Start"） |
| `Command` | string | コマンド名（例: "Text", "Jump"） |
| `Speaker` | string | 話者名（例: "カイリ"） |
| `Text` | string | 表示テキスト |
| `Args` | Dictionary<string, string> | 汎用パラメータ辞書 |

### メソッド

| メソッド | 戻り値 | 説明 |
|---------|-------|------|
| `GetString(key, default)` | string | 文字列パラメータを取得 |
| `GetFloat(key, default)` | float | 数値パラメータを取得 |
| `GetInt(key, default)` | int | 整数パラメータを取得 |
| `GetBool(key, default)` | bool | 真偽値パラメータを取得 |
| `HasArg(key)` | bool | パラメータの存在チェック |

---

## ScenarioLoader.cs

### クラス概要

CSVファイルをパースして `List<ScenarioCommand>` を生成する静的クラス。

### メソッド

| メソッド | 戻り値 | 説明 |
|---------|-------|------|
| `LoadFromCSV(TextAsset)` | List<ScenarioCommand> | CSVを読み込んでコマンドリストを返す |
| `FindLabelIndex(commands, label)` | int | ラベルを検索してインデックスを返す（-1: 見つからない） |

### パース仕様

- **CSV形式**: 5列（Label, Command, Speaker, Text, Args）
- **Args形式**: `key=value; key=value; ...`
- **ダブルクォート対応**: `"text, with comma"` 内のカンマは無視

---

## ScenarioExecutor.cs

### クラス概要

コマンドを順次実行してシナリオを進行させる MonoBehaviour。

### Inspector参照

| フィールド | 型 | 説明 |
|-----------|-----|------|
| `mainText` | TextMeshProUGUI | メインテキスト表示エリア |
| `speakerText` | TextMeshProUGUI | 話者名表示 |
| `speakerPanel` | GameObject | 話者名パネル |
| `characterImage` | Image | 立ち絵表示（右固定） |
| `backgroundImage` | Image | 背景画像 |
| `fadeImage` | Image | フェード演出用 |
| `choiceButtonPrefab` | GameObject | 選択肢ボタンプレハブ |
| `scenarioCSV` | TextAsset | 実行するシナリオCSV |
| `inputActions` | InputActionAsset | Input System |

### 外部API（public）

| メンバ | 型 | 説明 |
|--------|-----|------|
| `IsRunning` | bool | シナリオ実行中か |
| `StartScenario(csv, label, onEnd)` | void | 外部からシナリオを開始 |
| `StopScenario()` | void | シナリオを中断 |

### 対応コマンド一覧

#### テキスト系

| コマンド | 説明 | Args |
|---------|------|------|
| `Text` | 話者付きテキスト表示 + 立ち絵自動表示 | `face=表情ID` |
| `TextFade` | テキストをフェードイン表示 | `time=秒数` |
| `TextWait` | テキスト表示後に待機して消える | `time=秒数` |
| `Monologue` | 話者なしモノローグ | `face=表情ID` |

#### フロー制御

| コマンド | 説明 | Args |
|---------|------|------|
| `Jump` | 指定ラベルへジャンプ | `jumpTo=ラベル名` |
| `Branch` | フラグに応じてジャンプ | `flag=フラグ名; jumpTo=ラベル名` |
| `SetFlag` | フラグを設定 | `flag=フラグ名; value=true/false` |
| `Choice` | 選択肢表示 | テキストと `jumpTo` で分岐 |
| `Wait` | 指定秒数待機 | `time=秒数` |
| `End` | シナリオ終了 | - |

#### 演出系

| コマンド | 説明 | Args |
|---------|------|------|
| `FadeOut` | 画面フェードアウト | `time=秒数; color=white/black` |
| `FadeIn` | 画面フェードイン | `time=秒数` |
| `BGM` | BGM再生 | `name=BGM名; vol=音量` |
| `SE` | SE再生 | `name=SE名; vol=音量` |
| `BGImage` | 背景画像切り替え | `image=画像名` |
| `CharHide` | 立ち絵を非表示 + テキストクリア | - |

## 使用例

### CSV例

```csv
Label,Command,Speaker,Text,Args
,BGImage,,,image=Town01
Ev_Start,Text,カイリ,おはよう、ナギ。,face=smile
,Text,ナギ,おはよう。,face=normal
,Choice,,,選択肢→Ev_Route_A
,Choice,,,別の選択肢→Ev_Route_B
Ev_Route_A,Text,カイリ,Aを選んだね。,
,End,,,
Ev_Route_B,Text,カイリ,Bを選んだね。,
,End,,,
```
