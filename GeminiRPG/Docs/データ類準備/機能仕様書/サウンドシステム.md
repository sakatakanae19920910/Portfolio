# サウンドシステム機能仕様書

**最終更新**: 2026-01-20

---

## 概要

ゲーム内の全音声（BGM, BGS, ME, SE）を一元管理するシングルトンクラス。

---

## ファイル構成

| ファイル | 役割 |
|---------|------|
| `SoundManager.cs` | 音声管理シングルトン |
| `BootLoader.cs` | ゲーム起動時の初期化 |

---

## SoundManager.cs

### クラス概要

音声の再生・停止・音量管理を行うシングルトン MonoBehaviour。  
`DontDestroyOnLoad` で永続化され、どのシーンからでもアクセス可能。

### アクセス

```csharp
SoundManager.Instance.PlayBGM("BGM名");
```

### 音声カテゴリ

| カテゴリ | 説明 | ループ | 同時再生 |
|---------|------|--------|---------|
| **BGM** | バックグラウンドミュージック | ○ | 1つ |
| **BGS** | 環境音（雨、風など） | ○ | 1つ |
| **ME** | ジングル（レベルアップなど） | × | 1つ |
| **SE** | 効果音 | × | 複数 |

### Inspector参照

| フィールド | 型 | 説明 |
|-----------|-----|------|
| `bgmSource` | AudioSource | BGM 再生用 |
| `bgsSource` | AudioSource | BGS 再生用 |
| `meSource` | AudioSource | ME 再生用 |
| `seSourcePool` | AudioSource[] | SE 再生用プール |
| `bgmClips` | List<AudioClip> | BGM クリップ一覧 |
| `bgsClips` | List<AudioClip> | BGS クリップ一覧 |
| `meClips` | List<AudioClip> | ME クリップ一覧 |
| `seClips` | List<AudioClip> | SE クリップ一覧 |

### 主要メソッド

#### BGM

| メソッド | 説明 |
|---------|------|
| `PlayBGM(name, loop)` | BGM 再生 |
| `PlayBGMWithFade(name, duration, loop)` | フェードイン再生 |
| `StopBGM()` | BGM 停止 |
| `StopBGMWithFade(duration)` | フェードアウト停止 |

#### BGS

| メソッド | 説明 |
|---------|------|
| `PlayBGS(name, loop)` | 環境音再生 |
| `PlayBGSWithFade(name, duration, loop)` | フェードイン再生 |
| `StopBGS()` | 環境音停止 |
| `StopBGSWithFade(duration)` | フェードアウト停止 |

#### ME / SE

| メソッド | 説明 |
|---------|------|
| `PlayME(name)` | ジングル再生（BGM 一時停止） |
| `PlaySE(name)` | 効果音再生 |
| `PlaySE(name, volume)` | 音量指定で効果音再生 |

#### 音量制御

| メソッド | 説明 |
|---------|------|
| `SetMasterVolume(volume)` | マスター音量設定（0.0〜1.0） |
| `SetBGMVolume(volume)` | BGM 音量設定 |
| `SetBGSVolume(volume)` | BGS 音量設定 |
| `SetMEVolume(volume)` | ME 音量設定 |
| `SetSEVolume(volume)` | SE 音量設定 |
| `SaveVolumeSettings()` | 音量設定を PlayerPrefs に保存 |
| `LoadVolumeSettings()` | 音量設定を PlayerPrefs から読み込み |

#### その他

| メソッド | 説明 |
|---------|------|
| `StopAllSounds()` | 全音声を停止 |

## BootLoader.cs

### クラス概要

ゲーム起動時の初期化処理を行う MonoBehaviour。  
Boot シーンに配置し、初期化完了後に次のシーンへ遷移。

### Inspector参照

| フィールド | 型 | 説明 |
|-----------|-----|------|
| `nextSceneName` | string | 遷移先シーン名（デフォルト: "Title"） |

### 処理フロー

1. `Start()` が呼ばれる
2. 常駐マネージャー（SoundManager 等）が初期化される
3. `SceneManager.LoadScene(nextSceneName)` で遷移

---

## シーン構成

```
BootScene
├── SoundManager (DontDestroyOnLoad)
└── BootLoader
```

SoundManager は `Awake()` で `DontDestroyOnLoad` され、以降のシーンでも存続。

```
Assets/Resources/Audio/
├── BGM/          # BGM音源
├── BGS/          # 環境音
├── ME/           # ジングル
└── SE/           # 効果音
    ├── UI/       # UI系SE
    ├── Battle/   # 戦闘SE
    └── Field/    # フィールドSE
```
