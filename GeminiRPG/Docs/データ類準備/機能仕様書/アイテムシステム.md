# アイテムシステム機能仕様書

**最終更新**: 2026-01-20

---

## 概要

武器・防具・消耗品・重要アイテムのデータを管理するクラス群。

---

## ファイル構成

| ファイル | 役割 |
|---------|------|
| `ItemData.cs` | アイテム基底クラス（ScriptableObject） |
| `WeaponData.cs` | 武器データ |
| `ArmorData.cs` | 防具データ |
| `ConsumableData.cs` | 消耗品データ |
| `KeyItemData.cs` | 重要アイテムデータ |
| `ItemImporter.cs` | CSV → ScriptableObject 変換（Editor） |

---

## ItemData.cs（基底クラス）

### 共通フィールド

| フィールド | 型 | 説明 |
|-----------|-----|------|
| `itemId` | int | アイテムID |
| `itemName` | string | 表示名 |
| `description` | string | 説明文 |
| `buyPrice` | int | 購入価格（0=購入不可） |
| `sellPrice` | int | 売却価格（0=売却不可） |

---

## WeaponData.cs

### ID範囲

1001〜1999

### 固有フィールド

| フィールド | 型 | 説明 |
|-----------|-----|------|
| `weaponType` | WeaponType | 武器種 |
| `physicalAttr` | PhysicalAttribute | 物理属性（斬/刺/打） |
| `attack` | int | 攻撃力 |
| `elements` | Element[] | 元素属性 |
| `raceBonus` | Race[] | 種族特攻 |

### 武器種（WeaponType）

| 値 | 日本語 | 固有効果 |
|---|--------|---------|
| Dagger | 短剣 | 回避無視 |
| Sword | 片手剣 | 会心率UP |
| Club | 棍 | 混乱付与 |
| Wand | 片手杖 | 全体属性攻撃 |
| Spear | 槍 | 先制攻撃 |
| GreatSword | 両手剣 | ひるみ効果 |
| Hammer | 大槌 | 防御無視 |
| Staff | 両手杖 | 全体属性攻撃 |

## ItemImporter.cs（Editor）

### 機能

CSVファイルから ScriptableObject を自動生成するエディタ拡張。

### 使用方法

Unity メニュー: `Tools > Import Items from CSV`

### 対象CSV

- `Resources/MasterWeapon.csv`
- `Resources/MasterArmor.csv`
- `Resources/MasterConsumable.csv`
- `Resources/MasterKeyItem.csv`
