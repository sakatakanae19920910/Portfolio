# ステータス仕様書

## 概要

キャラクターのステータスシステムの設計仕様。

---

## ステータス一覧

### 成長するステータス（レベルアップで上昇）

| ステータス | 説明 | 用途 |
|-----------|------|------|
| HP | 体力 | 0になると戦闘不能 |
| MP | 魔力ポイント | スキル使用時に消費 |
| 攻撃力 | 物理攻撃力 | 物理ダメージ計算に使用 |
| 魔力 | 魔法攻撃力 | 魔法ダメージ計算に使用 |

### 固定ステータス（装備依存）

| ステータス | 説明 | 用途 |
|-----------|------|------|
| 防御力 | 物理防御力 | 被物理ダメージ軽減 |
| 抗魔力 | 魔法防御力 | 被魔法ダメージ軽減 |
| 敏捷性 | 素早さ | 行動順、命中計算に使用 |
| 狙われ率 | ターゲット選択の重み | 100が基準。高いほど狙われやすい |

### 戦闘中のみ変動するステータス

| ステータス | 説明 | 用途 |
|-----------|------|------|
| 回避値 | 装備回避 + バフ回避 | 命中率から減算される |

## ステータス計算式

### 最終ステータス

```
MaxHP = baseHP + equipHP
MaxMP = baseMP + equipMP
Attack = (baseAttack + equipAttack) × buffAttack
Defense = (baseDefense + equipDefense) × buffDefense
Magic = (baseMagic + equipMagic) × buffMagic
MagicDefense = (baseMagicDefense + equipMagicDefense) × buffMagicDefense
Speed = (baseSpeed + equipSpeed) × buffSpeed
Evasion = equipEvasion + buffEvasion
```

### レベルアップ時の成長

```
baseHP = 初期HP + growthHP × (level - 1)
baseMP = 初期MP + growthMP × (level - 1)
baseAttack = 初期攻撃力 + growthAttack × (level - 1)
baseMagic = 初期魔力 + growthMagic × (level - 1)
```

※防御力・抗魔力・敏捷性は成長しない（装備で強化）

## 命中・回避システム

### 命中計算式

```
中間命中率 = 100 - (100 - スキル命中) × (相手敏捷 / 自敏捷 + 1) / 2
最終命中率 = 中間命中率 - 相手のEvasion
命中判定 = 乱数(0-99) < 最終命中率
```

### ポイント

- **敏捷性**: 攻撃側は命中UP、防御側は相手の命中DOWNに作用
- **回避値**: 純粋に命中率を下げる（装備・バフで変動）

### 計算例

```
スキル命中90、自敏捷50、相手敏捷65、相手Evasion10 の場合

中間命中率 = 100 - 10 × (65/50 + 1) / 2
           = 100 - 10 × 1.15
           = 88.5%

最終命中率 = 88.5 - 10 = 78.5%
```

## ペチの変身システム

ペチは戦闘中に「猫モード」と「黒豹モード」を切り替えられる。

| モード | 特徴 |
|--------|------|
| 猫モード | 敏捷S、盗みや回避UP等のトリッキーな技 |
| 黒豹モード | HP/攻撃がS、戦闘力が高い |

### 実装方式

- CharacterDataを2つ作成（`Peti_Cat.asset`, `Peti_Panther.asset`）
- 変身時にCharacterDataを差し替える

## バフ・デバフ

### バフ補正（倍率）

| 対象 | デフォルト値 | 例 |
|------|-------------|-----|
| buffAttack | 1.0 | 1.5で攻撃50%UP |
| buffDefense | 1.0 | 0.5で防御50%DOWN |
| buffMagic | 1.0 | - |
| buffMagicDefense | 1.0 | - |
| buffSpeed | 1.0 | - |

### 回避バフ（加算）

| 対象 | デフォルト値 | 例 |
|------|-------------|-----|
| buffEvasion | 0 | 10で回避+10 |

### リセットタイミング

- 戦闘終了時に `ResetBuffs()` でデフォルト値に戻る
