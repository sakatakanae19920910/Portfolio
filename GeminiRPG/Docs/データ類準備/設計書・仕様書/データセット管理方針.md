# データセット管理方針

## 概要

本ドキュメントは『流転のジェミニ』で必要なデータセットの一覧と、それぞれの推奨管理方法をまとめたものである。

### 管理方法の基本方針

| 管理方法 | 使用ケース |
|----------|------------|
| **ScriptableObject** | Unityアセット（画像・音声・Animator等）への参照が必要なデータ |
| **CSV** | 大量のデータ、外部エディタ（Excel等）で編集したいデータ、テキスト中心のデータ |

## 1. キャラクター関連

| データ | 内容 | 推奨管理 | 理由 |
|--------|------|----------|------|
| プレイアブルキャラ基本 | ID, 名前, ステータス(SABCD), 成長率, 弱点/耐性, 種族 | **ScriptableObject** | 顔グラ・戦闘スプライト参照あり |
| キャラ口調設定 | 一人称, 二人称, 語尾 | CSV or JSON | テキストのみ、AI参照用 |
| 敵キャラ（通常） | ID, 名前, 種族, 弱点/耐性, スキル, 出現場所 | **CSV** | 数が多い、画像は種族ごとで対応可 |
| 敵キャラ（ボス） | 同上 + 特殊行動パターン | **ScriptableObject** | 専用グラフィック・演出参照あり |

## 2. スキル関連

| データ | 内容 | 推奨管理 | 理由 |
|--------|------|----------|------|
| アクティブスキル（魔法/技） | ID, 名前, カテゴリ, 属性, 対象, 効果, 消費MP, 習得Lv | **CSV** | 大量（100種以上）、エフェクトはID参照で対応 |
| パッシブスキル（加護） | ID, 名前, 効果, 解放条件(かけら数) | **CSV** | 数が少ない、テキストのみ |

### スキルカテゴリ
- 魔法（攻撃/回復/バフ/デバフ/状態異常）
- 技（物理攻撃/特殊）
- 加護（パッシブ）

## 3. アイテム関連

| データ | 内容 | 推奨管理 | 理由 |
|--------|------|----------|------|
| 武器 | ID, 名前, カテゴリ, 属性, 攻撃力, 特殊効果, 説明文 | **CSV** | 約60種類、アイコンはカテゴリごと |
| 防具 | ID, 名前, カテゴリ, 防御力, 耐性, 説明文 | **CSV** | 同上 |
| 消耗品 | ID, 名前, カテゴリ, 効果, 価格, 説明文 | **CSV** | 同上 |
| 重要アイテム | ID, 名前, 効果, フラグ | **CSV** | 数が少ない |
| アイコンマッピング | カテゴリ → Sprite | **ScriptableObject（1つ）** | 画像参照用 |

### 武器カテゴリ（8種）
| カテゴリ | 特徴 |
|----------|------|
| 短剣 | 回避無視 |
| 片手剣 | 会心率が高い |
| 棍 | 混乱付与 |
| 片手杖 | 全体属性攻撃（MP消費） |
| 槍 | 先制攻撃 |
| 両手剣 | ひるみ効果、敏捷性微減 |
| 大槌 | 防御無視、敏捷性減少 |
| 両手杖 | 全体攻撃（MP消費） |

### 防具カテゴリ（5種）
- 盾
- 軽装
- 重装
- 霊装（魔法耐性）
- 補助防具（マント/グローブ/兜/ブーツ）
- 装飾品

### 消耗品カテゴリ
- 回復系（HP/MP/戦闘不能/状態異常）
- 感情付与系（怒り/哀しみ）
- 戦闘系（属性ダメージ/耐性付与）
- 強化系（ステータス永続上昇）

## 7. システム関連

| データ | 内容 | 推奨管理 | 理由 |
|--------|------|----------|------|
| 感情システム | 感情タイプ, 効果倍率, 発動条件 | **CSV** | 4種類のみ |
| 状態異常 | ID, 名前, 効果 | **CSV** | 10種程度 |
| 属性相性 | 属性 → 倍率テーブル | **CSV** | 6x6マトリックス |
| 種族特攻 | 種族 → 特攻武器リスト | **CSV** | シンプル |
| エンパシー（好感度） | 上昇条件, エンディング閾値 | CSV or コード内定数 | 少量 |

### 感情システム
| 感情 | 効果 |
|------|------|
| 喜び | 全能力+5% |
| 怒り | 攻撃+15% |
| 哀しみ | 魔力+15% |
| 恐怖 | 攻撃・魔力-15%（獣のみ回避率1.5倍） |

### 状態異常一覧
- 毒/猛毒
- 麻痺
- 暗闇
- 睡眠
- 混乱
- 暴走
- 呪い
- 転倒/ひるみ
- 恐怖

### 属性一覧（6種）
- 炎・水・風・地・光・闇

### 物理属性（3種）
- 斬撃・刺突・打撃

### 種族一覧
- 人・獣・悪魔・精霊・龍・魚・草・飛行・石・鉄・不定形・鏡

## まとめ: 管理方法の棲み分け

### ScriptableObject（Unity参照が必要なもの）

```
ScriptableObject/
├── Character/
│   ├── Nagi.asset
│   ├── Mizuki.asset
│   ├── Pechi.asset
│   ├── Kaguya.asset
│   └── Hayate.asset
├── Boss/
│   ├── Boss01_Kaguya.asset
│   ├── Boss02_BanditLeader.asset
│   └── ...（12体）
├── Dungeon/
│   ├── Dungeon01_YuragiForest.asset
│   └── ...（10箇所）
└── IconMapping/
    └── ItemIconTable.asset
```

### CSV（大量データ・外部編集したいもの）

```
Resources/
├── Enemy/
│   ├── EnemyData.csv          # 通常敵（50種程度）
│   └── BossData.csv           # ボス敵
├── Skill/
│   ├── ActiveSkill.csv        # アクティブスキル（100種以上）
│   └── PassiveSkills.csv      # 加護
├── Item/
│   ├── MasterWeapon.csv       # 武器（60種程度）
│   ├── MasterArmor.csv        # 防具（60種程度）
│   ├── MasterConsumable.csv   # 消耗品（30種程度）
│   └── MasterKeyitem.csv      # 重要アイテム
├── Shop/
│   ├── DrugStore.csv          # 薬屋
│   ├── WeaponShop.csv         # 武器屋
│   └── ExchangeShop.csv       # 交換所
├── Scenario/
│   ├── Chapter1.csv
│   ├── Chapter2.csv
│   ├── ...
│   ├── PartyTalk.csv          # 仲間会話
│   ├── NPCTalk.csv            # NPC会話
│   ├── Documents.csv          # 文書片
│   └── PartnerDiary.csv       # パートナー日記
├── System/
│   ├── Emotions.csv           # 感情システム
│   ├── StatusEffects.csv      # 状態異常
│   ├── ElementMatrix.csv      # 属性相性表
│   └── RaceEffective.csv      # 種族特攻
└── Present/
    └── PresentReactions.csv   # プレゼント反応
```

1. CSV のカラム定義を決める
2. ScriptableObject のクラス設計を行う
3. CSV → ランタイムデータへの変換処理を実装する
