## 8. 実装ノート

### 8.1 継承構造案

```csharp
// 基底クラス
public abstract class ItemBase
{
    public int Id { get; set; }
    public string Name { get; set; }
    public string Description { get; set; }
    public int BuyPrice { get; set; }
    public int SellPrice { get; set; }
    public bool IsSellable => SellPrice > 0;
}

// プレゼント可能アイテム用インターフェース
public interface IGiftable
{
    bool IsGiftable { get; set; }
    string[] GiftTargets { get; set; }
    int EmpathyValue { get; set; }
}

// 武器
public class WeaponData : ItemBase, IGiftable { ... }

// 防具
public class ArmorData : ItemBase, IGiftable { ... }

// 消耗品
public class ConsumableData : ItemBase, IGiftable { ... }

// 重要アイテム（プレゼント不可）
public class KeyItemData : ItemBase { ... }
```

### 8.2 複数値の区切り文字

- セミコロン「;」で区切る
- 例: `Fire;Water` → Element.Fire, Element.Water

### 8.3 空欄の扱い

- 数値列の空欄 → 0として扱う
- 文字列列の空欄 → nullまたは空文字として扱う
- enum列の空欄 → デフォルト値（Noneなど）
- bool列の空欄 → FALSEとして扱う
