# アクティブスキル仕様書

**ActiveSkill.csv データ定義・実装ガイド**

---

## 1. 概要

ActiveSkill.csv は、ゲーム内で使用されるすべてのアクティブスキル（魔法・技）のマスターデータを定義するファイルです。

本ファイルは以下の目的で使用されます：

- スキルの基本情報（名称、MP消費、説明文など）の定義
- スキル効果の計算パラメータ（威力、命中率、乱数幅など）の設定
- 特殊効果（盗み、変身、吸収など）のタイプとパラメータ指定
- キャラクターごとの習得レベル管理

---

## 2. CSV列定義

### 2.1 基本情報列

| 列名       | 型         | 説明                                           | 例                    |
| :--------- | :--------- | :--------------------------------------------- | :-------------------- |
| スキルID   | int        | スキルの一意識別子。キャラごとに1000番台で管理 | 5001, 6001, 7001      |
| 並び順     | int        | メニュー内での表示順序                         | 1, 2, 3...            |
| 習得キャラ | string     | スキルを習得するキャラクターID                 | Nagi, Mizuki, Pechi   |
| 習得レベル | int/string | 習得レベル。Eventの場合はイベント習得          | 1, 8, 20, Event       |
| 名称       | string     | スキルの表示名                                 | ルクス, ねこばば      |
| MP         | int        | 消費MP。0の場合はMP消費なし                    | 4, 10, 0              |
| 説明       | string     | スキルの説明文。【】内は要約                   | 【攻撃 光 単体小】... |

### 2.2 効果パラメータ列

| 列名     | 型     | 説明                        | 例                  |
| :------- | :----- | :-------------------------- | :------------------ |
| 対象範囲 | enum   | スキルの対象。詳細は後述    | OneEnemy, AllAllies |
| 命中率   | int    | 基本命中率（%）             | 95, 80, 100         |
| 効果量   | int    | 基本威力/回復量             | 30, 70, 140         |
| 乱数幅   | int    | 効果量の±変動幅             | 10, 15, 0           |
| 対象能力 | enum   | 効果を与える能力値          | HP, MP, Attack      |
| 属性     | string | スキル属性。複数は;で区切る | Light, Water;Dark   |
| 状態     | string | 付与/治療する状態異常       | Poison, Sleep       |
| 感情     | string | 付与する感情                | Fear                |

### 2.3 特殊効果列

| 列名       | 型     | 説明                       | 例                     |
| :--------- | :----- | :------------------------- | :--------------------- |
| 特殊効果   | enum   | 特殊効果タイプ。詳細は後述 | Heal, Steal, Transform |
| パラメータ | string | 特殊効果用パラメータ       | Duration:5             |

## 3. 対象範囲（TargetScope）

| 値               | 対象           | 説明                      |
| :--------------- | :------------- | :------------------------ |
| Self             | 自分           | 使用者自身のみ            |
| OneAlly          | 味方単体       | 生存している味方1体を選択 |
| AllAllies        | 味方全体       | 生存している味方全員      |
| OneDeadAlly      | 戦闘不能味方   | 戦闘不能の味方1体を選択   |
| AllDeadAllies    | 戦闘不能全体   | 戦闘不能の味方全員        |
| OneRandomAlly    | 味方ランダム   | 味方からランダムに1体     |
| OneEnemy         | 敵単体         | 敵1体を選択               |
| AllEnemies       | 敵全体         | 敵全員                    |
| OneRandomEnemy   | 敵ランダム     | 敵からランダムに1体       |
| AllRandomEnemies | 敵ランダム複数 | 敵全体にランダム回数攻撃  |

## 4. 特殊効果タイプ（EffectType）

### 4.1 基本タイプ

| タイプ      | 説明                            | パラメータ例 |
| :---------- | :------------------------------ | :----------- |
| (空)        | 通常攻撃/回復。標準列のみで処理 | -            |
| Heal        | HP/MP回復（明示用）             | -            |
| Revive      | 戦闘不能回復                    | -            |
| CureAilment | 状態異常治療。状態列で対象指定  | -            |

### 4.2 バフ/デバフタイプ

| タイプ     | 説明             | パラメータ例                     |
| :--------- | :--------------- | :------------------------------- |
| Buff       | 能力値上昇       | Duration:5                       |
| Debuff     | 能力値低下       | Duration:5                       |
| RandomBuff | ランダム能力上昇 | Stats:Attack;Defense\|Duration:3 |

### 4.3 ダメージ特殊タイプ

| タイプ             | 説明                   | パラメータ例                             |
| :----------------- | :--------------------- | :--------------------------------------- |
| ScaledDamage       | 特定能力依存ダメージ   | ScaleStat:Agility\|Ratio:0.5             |
| MultiHit           | 複数回攻撃             | MinHits:3\|MaxHits:6                     |
| HighCritical       | 高会心率攻撃           | CritRate:25                              |
| RandomDamage       | 乱数幅大（投石など）   | -                                        |
| DamageWithAilment  | ダメージ＋状態異常確率 | AilmentRate:30                           |
| DamageWithSelfBuff | ダメージ＋自己バフ     | BuffStat:Attack\|BuffRate:15\|Duration:3 |

### 4.4 特殊行動タイプ

| タイプ        | 説明                         | パラメータ例                                               |
| :------------ | :--------------------------- | :--------------------------------------------------------- |
| Steal         | アイテム盗み                 | SuccessRate:70                                             |
| Drain         | HP/MP吸収                    | DrainType:HP\|DrainRate:50                                 |
| Transform     | 変身（豹変など）             | FormID:BlackPanther\|Duration:4                            |
| Provoke       | 挑発                         | SelfFear:true\|Duration:3                                  |
| AddResist     | 属性耐性付与                 | Elements:Fire;Water;Wind;Earth\|Duration:5                 |
| EmpathyDamage | エンパシー依存（ラスボス用） | SelfDebuff:Defense;MagicDefense\|Duration:1\|BossOnly:true |

---

## 5. パラメータ書式

EffectParams列は以下の書式規則に従います：

- キーと値は「:」（コロン）で区切る
- 複数パラメータは「|」（パイプ）で区切る
- リスト値は「;」（セミコロン）で区切る

**例：**

```
Duration:5
Stats:Attack;Defense;MagicPower|Duration:3
SelfDebuff:Defense;MagicDefense|Duration:1|BossOnly:true
```

## 7. 実装ノート

### 7.1 ダメージ計算式

```
基本ダメージ = 効果量 ± 乱数幅
最終ダメージ = 基本ダメージ × 属性補正 × 能力補正
```

※ ScaledDamageの場合は追加で（参照能力値 × Ratio）を加算

### 7.2 状態異常付与確率

```
付与確率 = 命中率 × (1 - 対象の耐性)
```

※ DamageWithAilmentの場合はAilmentRateを追加で乗算

### 7.3 バフ/デバフの重複

- 同一能力への複数バフ：効果量を加算
- 同一スキルの再使用：Durationをリセット（効果量は加算しない）
- バフとデバフの併存：それぞれ独立して適用

### 7.4 豹変（Transform）の処理

- 変身中は専用ステータスを適用（BlackPanther形態）
- 使用可能スキルが変化（猫モード→黒豹モード）
- Duration経過で自動的に元の姿に戻る
- 変身中に戦闘不能→復活時は通常状態
