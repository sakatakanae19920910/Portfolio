施設フットプリント

  • 薬屋（パートナーの家） 6×7
    2階建て。1F：店舗＋リビング、2F：主人公の部屋＋パートナーの部屋。内部マップは別シーン推奨
  • 武器屋 8×6
    町で一番大きな建物。広場に面する。内部にNPC5人＋店主＋本棚
  • 酒場 6×5
    奥まった位置。内部に地下への隠し階段あり
  • 交換屋 4×4
    小さな店。ミタマ石との交換
  • 占い屋 3×4
    町の最奥。ひっそりと小さい。探さないと見つからない感を演出

Unity Tilemap レイヤー構成

  • 0Ground— 地面・草・土道
  • 1Path— 踏み固められた道
  • 2Objects— 岩・ベンチ・柵・樽
  • 3Buildings— 建物の壁・屋根
  • 4Rooftop— 屋根（進入時に透過）
  • 5Collision— 当たり判定（非表示）

技術メモ

Tilemap Collider 2D を Buildings レイヤーに設定し、Composite Collider 2D と組み合わせるとパフォーマンスが良くなります。

屋根の透過：建物に入った時に屋根タイルを非表示にするには、Rooftopレイヤーを別Tilemapにして、Trigger進入時にSpriteRenderer.colorのalphaを下げるのが簡単です。

素材探しのヒント：itch.io で「top-down tileset RPG」で検索すると無料〜安価な素材が多数見つかります。タイルサイズ（16/32/48px）を統一することが重要。
